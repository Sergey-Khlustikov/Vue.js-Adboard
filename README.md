# Vue.js Ad board SPA

> Demo ad board SPA where you can:
>  - make an account, post your own ads, edit them;
>  - take, filter and manage orders;
>  - view ads by other users and leave an application.

Project demo is available at https://ads-project-51b22.firebaseapp.com/

## What was used:

- Vuex
- Vue Router
- Vuetify for UI
- Firebase by Google

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
